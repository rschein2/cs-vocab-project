How do you start a new tmux session?
Command: tmux

Why: Creates a new unnamed session. If your SSH connection drops, the session keeps running in the background.

Better practice - name your sessions:

tmux new -s session-name

Why name sessions: Makes it easy to reconnect to the right session later, especially when you have multiple projects running.

Variations:

  • tmux new -s work - create session named "work"
  • tmux new -s project -n window1 - create session with named first window

You're in a tmux session and need to disconnect without killing it. What's the command?
Command: C-b d

What it does: d stands for "detach". Your session keeps running in the background with all processes intact.

Why this matters: You can close your terminal/SSH connection and everything keeps running. Reconnect later to pick up exactly where you left off.

Alternative ways to detach:

  • tmux detach - type this inside tmux (longer)
  • C-b D - choose which session to detach from a list

Pro tip: Make detaching a habit before closing SSH connections!

How do you reconnect to a detached tmux session?
Command: tmux attach

Or (shorthand): tmux a

Why: Attaches to the most recent session. All your windows, panes, and running processes are exactly as you left them.

If you have multiple sessions:

tmux attach -t session-name
# Or shorthand:
tmux a -t session-name

Workflow:

tmux ls                    # List all sessions
tmux attach -t work        # Attach to "work" session

Create or attach:

tmux new -As session-name

Attaches if exists, creates if doesn't - great for scripts!

How do you see a list of all running tmux sessions?
Command: tmux ls

Or: tmux list-sessions

Example output:

work: 3 windows (created Mon Jan 15 10:23:45 2024)
dev: 2 windows (created Mon Jan 15 11:45:12 2024) (attached)

From inside tmux:

C-b s

Shows interactive list where you can switch between sessions with arrow keys and Enter.

Why the interactive view is better: You can see window previews and switch sessions without detaching.

You're in a tmux session and want to split the current pane horizontally (top/bottom). What's the command?
Command: C-b "

Why the double-quote: Think of it as a horizontal line, splitting top from bottom.

Mnemonic: The " symbol looks like a horizontal divider when you squint!

Alternative (easier to remember):

Add to your ~/.tmux.conf:

bind - split-window -v

Then use C-b - (minus is horizontal!)

Pro tip: The new pane starts in the same directory as the current pane.

You're in a tmux session and want to split the current pane vertically (left/right). What's the command?
Command: C-b %

Why the percent sign: Think of it as a vertical line dividing left from right.

Mnemonic: The % symbol has a vertical line in the middle!

Alternative (easier to remember):

Add to your ~/.tmux.conf:

bind | split-window -h

Then use C-b | (pipe is vertical!)

Common setup:

# Intuitive splits
bind | split-window -h  # vertical split
bind - split-window -v  # horizontal split

How do you navigate between panes in tmux?
Command: C-b [arrow key]

Why: Arrow keys match the direction you want to move. Intuitive once you have multiple panes.

Examples:

  • C-b ← - move to left pane
  • C-b → - move to right pane
  • C-b ↑ - move to pane above
  • C-b ↓ - move to pane below

Cycle through panes:

  • C-b o - next pane (circular)
  • C-b ; - toggle between current and previous pane

Vim-style navigation:

Add to ~/.tmux.conf for hjkl movement:

bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

How do you create a new window in your current tmux session?
Command: C-b c

Why: c stands for "create". Each window is like a separate terminal tab, and can contain multiple panes.

Windows vs Panes:

  • Windows - separate screens (like browser tabs)
  • Panes - split views within a window (like split screen)

Create named window:

C-b c
# Then rename:
C-b ,
# Type new name

Or from command line inside tmux:

tmux new-window -n "window-name"

How do you switch between windows in tmux?
Commands:
  • C-b n - next window
  • C-b p - previous window
  • C-b [0-9] - jump to window by number

Interactive window list:

C-b w

Shows all windows across all sessions. Use arrows and Enter to select.

Why numbers matter: Windows are numbered starting at 0. You can see the number in the status bar at the bottom.

Examples:

  • C-b 0 - jump to window 0
  • C-b 1 - jump to window 1
  • C-b l - toggle to last window (like cd -)

How do you close/kill the current pane in tmux?
Simple way: exit or Ctrl+d

Why: Just exit the shell normally. When the shell exits, the pane closes.

Force kill (if process is stuck):

C-b x

Prompts for confirmation before killing the pane.

Important: If you kill the last pane in a window, the window closes. If you kill the last window in a session, the session closes.

Kill current window:

C-b &

Closes all panes in the current window (asks for confirmation).

How do you kill an entire tmux session from outside tmux?
Command: tmux kill-session -t session-name

Why: Terminates the session and all its windows/panes. Useful for cleaning up old sessions.

Kill current session (from inside):

tmux kill-session

Kill all sessions except current:

tmux kill-session -a

Kill all sessions:

tmux kill-server

Warning: This terminates EVERYTHING. Use with caution!

List then kill workflow:

tmux ls
tmux kill-session -t old-project

You're in tmux and need to scroll up to see previous output. How do you enter scroll/copy mode?
Command: C-b [

Why: Enters "copy mode" which lets you scroll through terminal history using keyboard.

Navigation in copy mode:

  • ↑↓ or j/k - move up/down line by line
  • Page Up/Down or C-b/C-f - move by page
  • g - go to top
  • G - go to bottom
  • / - search forward
  • ? - search backward

Exit copy mode: q or Esc

Pro tip: Enable mouse mode for scrolling with mouse wheel:

set -g mouse on

Add this to ~/.tmux.conf

How do you rename an existing tmux session?
From inside the session: C-b $

Why: Opens a prompt at the bottom where you can type the new name.

From outside tmux:

tmux rename-session -t old-name new-name

Rename current window:

C-b ,

Opens prompt to rename the current window.

Why naming matters: When you have multiple sessions, names like "work", "personal", "project-x" are much clearer than "0", "1", "2".

Auto-naming windows:

Disable auto-rename to keep your custom names:

set-option -g allow-rename off

How do you resize a tmux pane?
Command: C-b C-[arrow]

Why hold Ctrl: You can press the arrow multiple times while holding Ctrl+b to resize incrementally.

Examples:

  • C-b C-↑ - resize pane upward (make taller)
  • C-b C-↓ - resize pane downward (make shorter)
  • C-b C-← - resize pane left (make narrower)
  • C-b C-→ - resize pane right (make wider)

Alternative - manual mode:

C-b :
resize-pane -D 5    # Down 5 lines
resize-pane -U 5    # Up 5 lines
resize-pane -L 5    # Left 5 columns
resize-pane -R 5    # Right 5 columns

Even out pane sizes:

C-b M-1    # Even horizontal split
C-b M-2    # Even vertical split

How do you zoom/maximize a single pane in tmux (temporarily hide other panes)?
Command: C-b z

Why: Toggles the current pane to full-screen. Other panes are hidden but not closed.

Use case: You have a 4-pane layout but need to focus on one pane's output or edit a long file. Zoom in, do your work, then zoom out to return to the split layout.

Exit zoom: Press C-b z again (it's a toggle).

Visual indicator: When zoomed, you'll see a Z indicator in the window status bar.

Pro tip: This is way better than closing and recreating panes!

How do you access the tmux command prompt?
Command: C-b :

Why: Opens a command prompt where you can type any tmux command directly (like : in vim).

Example commands:

:set -g mouse on                    # Enable mouse mode
:split-window -h                    # Split vertically
:rename-session new-name            # Rename session
:resize-pane -D 5                   # Resize down 5 lines
:source-file ~/.tmux.conf           # Reload config

Why use the prompt: Some commands don't have keyboard shortcuts, or you forget the shortcut. The prompt gives you access to the full tmux API.

Tab completion: Works in the command prompt! Start typing and press Tab.

How do you reload your tmux configuration file without restarting tmux?
From inside tmux: 
C-b :
source-file ~/.tmux.conf

Or create a keybinding:

Add to ~/.tmux.conf:

bind r source-file ~/.tmux.conf \; display "Config reloaded!"

Then just press C-b r to reload!

Why this matters: You can tweak your tmux config and test changes immediately without detaching/reattaching or restarting sessions.

From outside tmux:

tmux source-file ~/.tmux.conf

Note: Some settings (like prefix key changes) might need a full restart to work properly.

What's the difference between detaching from a tmux session and killing it? When would you use each?
Detach (C-b d):
  • Session keeps running in background
  • All processes continue
  • You can reattach later
  • Like minimizing a window
Kill (tmux kill-session):
  • Session terminates completely
  • All processes are killed
  • Cannot reattach
  • Like closing a window

When to detach:

  • Long-running processes (builds, downloads, servers)
  • You're logging out but want to continue work later
  • SSH connection is unstable

When to kill:

  • You're completely done with the session
  • Cleaning up old/unused sessions
  • Freeing up resources

Best practice: Detach by default, kill when you're sure you're done.
