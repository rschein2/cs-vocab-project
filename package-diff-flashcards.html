You need to install a package called "nodejs" on your Fedora system. How do you install it? What if you want to install it without asking for confirmation?
Install package: sudo dnf install nodejs
Without confirmation: sudo dnf install -y nodejs

Why: dnf is the package manager for Fedora/RHEL systems. The -y flag automatically answers "yes" to prompts, useful for scripts.

Alternatives:

  • sudo apt install nodejs (Debian/Ubuntu)
  • sudo yum install nodejs (older RHEL systems)

You need to create a patch file from differences between two files to share with someone. How do you generate a patch? How would they apply it?
Create patch: diff -u original.txt modified.txt > changes.patch
Apply patch: patch original.txt < changes.patch

Why: -u creates unified diff format (standard patch format). The patch file can be applied to the original to recreate the modified version.

You want to compare two files but only care if they're different, not what the differences are. How do you check if two files are identical? What about binary files?
Silent comparison: cmp -s file1 file2 && echo "Same" || echo "Different"
Show first difference: cmp file1 file2

Why: cmp compares files byte-by-byte and exits on first difference, faster than diff. -s flag makes it silent (only sets exit code). Works on binary files.

Alternatives:

  • diff -q file1 file2 (brief, only reports if different)
  • cmp -l file1 file2 (list all differences, byte positions)
  • md5sum file1 file2 (compare checksums)

Exclude from updates: Add exclude=packagename to /etc/dnf/dnf.conf

Why: Excluding packages prevents automatic updates, useful for packages you've customized or that have known issues. Per-command exclusion doesn't persist.
