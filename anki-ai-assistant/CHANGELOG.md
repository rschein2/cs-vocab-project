# Changelog

All notable changes to AI Card Assistant will be documented in this file.

## [1.0.0] - 2025-11-18

### Added
- Initial release
- AI-powered card refinement during review
- Support for Anthropic Claude API
- 8 predefined actions (simplify, add example, clarify, create related card, etc.)
- Custom question mode
- Settings dialog for API key configuration
- Cost estimation display
- Preview before applying changes
- Auto-tagging of AI-generated cards
- Keyboard shortcut (Ctrl+Shift+A)
- Append or replace modes for card modification
- API key testing functionality

### Features
- Works with any Anki deck
- Context-aware (uses deck name and tags)
- Supports Claude Sonnet 4, Claude 3.5 Sonnet, and Claude 3 Opus
- Configurable max tokens
- User provides own API key (no subscription needed)
- Clean, modern UI with gradient buttons

### Documentation
- Comprehensive README with installation guide
- Usage examples
- FAQ section
- Troubleshooting guide
- Cost information
